TrackPro API Documentation

Welcome to the TrackPro GPS Tracking API. Our REST API allows you to integrate real-time GPS tracking, geofencing, and alert management into your applications.

Base URL

https://api.trackpro.com/v1

Getting Started

To use the TrackPro API, you'll need to:

  1. Sign up for a TrackPro account
  2. Generate an API key from your dashboard
  3. Make authenticated requests to our endpoints

Authentication

All API requests must be authenticated using an API key. Include your API key in the Authorization header:

curl https://api.trackpro.com/v1/devices \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

Security Warning

Never share your API keys publicly or commit them to version control. Keep them secure!

Devices API

GET /devices

List All Devices

Returns a list of all devices associated with your account.

Example Request
curl https://api.trackpro.com/v1/devices \
  -H "Authorization: Bearer YOUR_API_KEY"
Example Response
{
  "success": true,
  "data": [
    {
      "id": "dev_1a2b3c4d",
      "name": "Tesla Model 3",
      "type": "vehicle",
      "status": "online",
      "battery": 85,
      "last_update": "2025-10-23T13:27:00Z",
      "location": {
        "latitude": 37.7749,
        "longitude": -122.4194,
        "speed": 45,
        "heading": 180
      }
    }
  ],
  "total": 4,
  "page": 1
}
GET /devices/:id

Get Device Details

Retrieve detailed information about a specific device.

POST /devices

Create Device

Register a new GPS tracking device.

Request Body
{
  "name": "My Vehicle",
  "type": "vehicle",
  "imei": "123456789012345",
  "description": "Company delivery van"
}
DELETE /devices/:id

Delete Device

Remove a device from your account.

Tracking API

GET /tracking/:device_id/location

Get Real-Time Location

Get the current location of a device.

Example Response
{
  "success": true,
  "data": {
    "device_id": "dev_1a2b3c4d",
    "timestamp": "2025-10-23T13:27:00Z",
    "location": {
      "latitude": 37.7749,
      "longitude": -122.4194,
      "altitude": 52,
      "accuracy": 8
    },
    "movement": {
      "speed": 45,
      "heading": 180,
      "status": "moving"
    },
    "battery": 85,
    "signal_strength": "strong"
  }
}
GET /tracking/:device_id/stream

WebSocket Stream

Subscribe to real-time location updates via WebSocket.

const ws = new WebSocket('wss://api.trackpro.com/v1/tracking/dev_1a2b3c4d/stream?token=YOUR_API_KEY');

ws.onmessage = (event) => {
  const location = JSON.parse(event.data);
  console.log('New location:', location);
};

Geofences API

POST /geofences

Create Geofence

Create a virtual boundary for location-based alerts.

Request Body
{
  "name": "Office Zone",
  "type": "circle",
  "center": {
    "latitude": 37.7749,
    "longitude": -122.4194
  },
  "radius": 500,
  "devices": ["dev_1a2b3c4d"],
  "alerts": {
    "on_enter": true,
    "on_exit": true
  }
}
